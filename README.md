# 物流仿真设计项目

> [!info] 项目概述
> 本项目是一个基于 RaLC（Rapid Virtual Model Builder For Logistics Center Verification）系列的物流仿真模型库，包含多个应用项目和可复用的设备模板。所有文档采用 Obsidian Flavored Markdown 格式编写，兼容 GitHub 和 Gitee 平台。

---
## 使用方法
### 克隆仓库
```
#克隆仓库（包含LFS文件）
git clone https://github.com/GBGBGBD/lelong-models.git

#或从Gitee克隆
git clone https://gitee.com/GBGBGBD/lelong-models.git
```

### 打开方式
将 `.emu`文件托到==RaLC==系列物流仿真软件图标即可打开
